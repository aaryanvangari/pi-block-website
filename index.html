No affiliation with Pi-Hole

Pi Block is an independent open-source project created by developers who wanted better control over their networks. It is not officially affiliated with or endorsed by the Pi-hole project. Pi Block works as a client for Pi-hole servers and respects all the principles that make Pi-hole valuable, but it is a separate application built and maintained independently. We are grateful for the Pi-hole community and the foundation it provides for network-level ad blocking and DNS management.

Frequently Asked Questions

What is Pi Block?

Pi Block is a mobile client for Pi-hole, built with Flutter. It lets you monitor and control your Pi-hole DNS sinkhole from your phone or tablet, giving you real-time visibility into your network's DNS queries and the ability to manage blocking rules on the go.

Does Pi Block collect data?

No. Pi Block has no tracking, no analytics, and no data collection. All communication happens between your device and your own Pi-hole instance. Nothing is sent to external servers or stored anywhere except on your network.

Which platforms are supported?

Pi Block currently supports Android and iOS. Web, Linux, and Windows versions are planned for future releases. You can follow development progress on GitHub.

Is Pi Block free?

Yes, Pi Block is completely free. It is open-source software with no ads, no paywalls, and no forced monetization. Donations are optional and help support ongoing development.

How do I contribute?

Pi Block is open-source and welcomes contributions. Visit the GitHub repository to report issues, suggest features, or submit code. You can also support the project by donating or sharing it with others.
