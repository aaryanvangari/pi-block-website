Procedure

Web

Preferred method and working method for opting into closed testing

  1. Sign in using your email in Chrome. If using a different account, sign out first and sign in with the email which is added to the closed testing track.
  2. Open the web link from above in Chrome.
  3. The website will ask you to become a tester, accept it.
  4. If step 3 doesn't appear, your email may not be added to the closed testing track. Please request for access.
  5. Click the link similar to "download the Pi-Block app" from the web page.
  6. Install the app (approximately 11 MB; takes 5-10 seconds).
  7. You can use any Pi-Hole credentials, but below are provided for convenience.
  8. Use the following mock credentials:
    Mock URL
    https://mock-pihole.opensquares.in
    Mock API Token
    test-token
  9. Log in to the app and explore all pages.

Google Play

  1. Sign in using your account in Google Play.
  2. If it's not your email which is added to the closed testing track, log out from Google Play.
  3. This may be inconvenient for daily usage.
  4. Once logged in with the correct account, follow the Web procedure from step 2 using the Google Play link.
  5. This method is not fully tested, so steps may vary.

Notes

  1. Since this uses mock data, not all functions and interactions with app will work.
  2. On a real live Pi-Hole server, everything should function properly.
